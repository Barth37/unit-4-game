Crystal Collector Game



  • You will be given a random number at the start of the game.

  • There are four crystals below. By clicking on a crystal, you will add a specific amount of points to your total score.

  • You win the game by matching your total score with the random number below. If you exceed the random number, you lose.

  • The value of each crystal is hidden so you have to figure out the value of each by clicking on them.

  • Each time the game restarts, the values of the gems change.
